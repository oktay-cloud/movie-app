# Movie App 🎬

A modern React web application to browse movies, add favorites, and view detailed information.

---

## Features

- Browse movies on the Home page
- Add and remove favorites in real-time ❤️
- Favorites are saved using LocalStorage
- View movie details including title, year, and trailer
- Back button for easy navigation
- Responsive UI with smooth hover effects

---

## Technologies Used

- React
- React Router
- LocalStorage
- CSS
